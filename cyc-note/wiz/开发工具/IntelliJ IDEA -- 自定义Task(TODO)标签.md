### [IntelliJ IDEA -- 自定义Task(TODO)标签](http://b-l-east.iteye.com/blog/2209418)

- [IntelliJ](http://b-l-east.iteye.com/category/339112)

Eclipse下可以添加类似于TODO, FIXME, XXX 这类的自定义任务标签

在IntelliJ下也可以如此：

preference -> Editor -> TODO

发现已有两个标签:

\btodo\b.*  ---- 对应TODO

\bfixme\b.* ---- 对应FIXME

点加号可以添加自定义的TODO标签，如：

\bxxx\b.*  ---- 对应XXX

有三个图标可选择，自己随意选择。

要查看TODO标签，在软件下方的『TODO』窗口中，点击漏斗按钮可以筛选标签。
