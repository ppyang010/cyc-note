## freemarker编辑器eclipse插件

freemarker编辑器eclipse插件

支持语法高亮,语法校验,代码提示的工具

freemarker  IDE(JBoss):

http://download.jboss.org/jbosstools/updates/development/

安装方法:

eclipse --> Help --> Install New Software --> Add --> Name=freemarker,Location=http://download.jboss.org/jbosstools/updates/development/

myeclipse --> Help --> Software Updates --> Find and Install --> Search for new features to install --> Add --> Name=freemarker,Location=http://download.jboss.org/jbosstools/updates/development/

只需选"JBoss Tools - 3.1.0.v200912250601M-H198-CR1"下的"FreeMarker IDE"即可,其他工具不用到就不要下载,以节省安装时间。

eclipse中安装freemarker插件及ftl使用freemarker[编辑器](http://www.07net01.com/tags-%E7%BC%96%E8%BE%91%E5%99%A8-0.html)

在线安装的方法是：Help –> Install New [software](http://www.07net01.com/tags-software-0.html) 点击  and

http://download.jboss.org/jbosstools/[update](http://www.07net01.com/tags-update-0.html)s/development/indigo/
